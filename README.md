# Ready‑Reckoners

A personal knowledge base of quick reference notes, cheat sheets, and study materials across software development, cloud, and data technologies.  
This repository is designed as a **ready reckoner** — fast access to essential commands, concepts, and workflows for learning, interviews, and daily development.

---

## Contents

- **Appdev.txt** – Application development notes  
- **Cloud Computing.txt** – Cloud fundamentals and architectures  
- **CSS.xlsx** – CSS reference in spreadsheet format  
- **Docker.txt / Dockerfile_example.txt** – Docker usage and examples  
- **GenAI/** – Generative AI notes and experiments  
- **git.txt** – Git commands and workflows  
- **JavaScript Notes**  
  - `JS.txt` – General notes  
  - `JS_Notes/` – Structured notes  
  - `JS_Chapterwise_Notes.pdf` – Chapter‑wise reference  
- **Kubernetes.txt** – Kubernetes concepts and commands  
- **linux.txt** – Linux essentials and commands  
- **Microservices.txt** – Microservices architecture notes  
- **MongoDB-Mongoose.txt** – MongoDB and Mongoose ORM notes  
- **Node-Express-Pug.txt** – Node.js, Express, and Pug templating notes  
- **numpy.txt / pandas.txt** – Python data science libraries  
- **Python.txt** – Python language reference  
- **React.txt** – React.js notes  
- **SQL-Django-ORM.txt** – SQL and Django ORM notes  
- **serverless_responsibility_model.png** – Serverless responsibility model diagram  
- **tailwind.txt** – Tailwind CSS notes  

---

## Purpose

- Provide **quick access** to essential commands, concepts, and workflows.  
- Act as a **ready reckoner** for interview prep, project work, and daily development.  
- Maintain a **modular knowledge base** that can be extended over time.
